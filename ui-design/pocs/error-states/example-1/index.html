Error State Example 1: Inline Error Tooltips

This mockup demonstrates how error states can be shown using inline tooltips that appear when hovering over gates with errors. Each problematic gate shows a red error indicator badge with an exclamation mark. When the user hovers over the gate, a tooltip appears with a detailed explanation of the error.

Hover over gates with red badges to see error messages.
