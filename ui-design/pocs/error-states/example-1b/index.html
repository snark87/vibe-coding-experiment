Error State Example 1B: Inline Tooltips with Visual Highlighting

This mockup demonstrates a hybrid approach to error visualization, combining inline tooltips with broader visual highlighting. Each problematic gate shows a red error indicator badge, and problematic areas are highlighted with a light red background. When the user hovers over a gate with an error, a detailed tooltip appears with a specific explanation.

Hover over gates with red badges to see error messages. Note how the error areas are highlighted in red to provide context about problem locations.
